ОБУЧАЮЩЕЕ ВИДЕО

Цель: создать базу данных школьной библиотеки, в которой можно:

  • учитывать книги,
  • регистрировать читателей,
  • отслеживать, кто взял какую книгу и когда её нужно вернуть.

Задание в Access:

1. Создать базу данных

Создайте новый файл базы данных с названием:
АРМ_Библиотека.accdb

Связи:

  • ID_читателя — внешний ключ к таблице Читатели
  • ID_книги — внешний ключ к таблице Книги

3. Ввести данные

  • В таблицу Книги — не менее 5 книг разных жанров (например, сказки, фантастика, детектив и т.д.)
  • В таблицу Читатели — 3–5 учеников (с указанием класса и даты регистрации)
  • В таблицу Выдачи — по 2–3 записи, кто какую книгу взял и когда

4. Создать запросы:

1.    Найти всех читателей, которые брали книги определённого жанра

2.    Вывести список всех книг, выданных после определённой даты

3.    Показать, какие книги на руках у читателя по фамилии

5. Создать форму:

Создайте форму для ввода новых читателей (форма для таблицы Читатели), чтобы библиотекарю было удобно регистрировать новых учеников.

6. Создать отчёт:

Создайте отчёт "Книги на руках", в котором будет отображаться:

  • ФИО читателя,
  • Название книги,
  • Дата выдачи,
  • Ожидаемая дата возврата.

Вопросы для обсуждения после задания:

  1. Почему библиотекарю удобно использовать такую базу данных?
  2. Что может случиться, если вести учёт только на бумаге?
  3. Как вы думаете, можно ли связать такую базу с сайтом библиотеки?
